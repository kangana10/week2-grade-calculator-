# Student Grade Calculator

## Project Description

The Student Grade Calculator is a Python-based project developed using control flow, loops, lists, functions, and exception handling concepts.

This program allows users to:
- Add multiple student records
- Enter marks for different subjects
- Calculate averages and grades automatically
- Generate personalized comments
- View class statistics
- Search for students
- Save results into a text file

This project was created as part of the Python Developer Internship - Week 2 Assignment.
